# Live Nepali Date Time

### Getting started

Include this nepali.js in a project and create HTML elements with specified ids, and you are good to go!

२०७६ असार २७, शुक्रबार  
2076 Asar 27, Friday  
१७:३१:०७  
17:31:07

#### Setup Guide

1. Add nepali.js to your project and include nepali.js on your page.
2. Add an HTML element with one of the following ids  
   DATE_IN_NEPALI  
   DATE_IN_ENGLISH  
   TIME_IN_NEPALI  
   TIME_IN_ENGLISH
3. Add the ```updateClock()``` method on onload attribute of your body.

#### Done !
